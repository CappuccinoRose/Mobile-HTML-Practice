# Mobile-HTML-Practice

这是一个移动端HTML/CSS/JS练习项目，包含多个独立的小练习，涵盖布局、交互、响应式等前端核心技能。


## 技术栈

- **HTML5**：语义化标签、响应式meta设置
- **CSS3**：Flex布局、REM适配、渐变、动画、媒体查询
- **JavaScript**：原生JS、jQuery（部分项目）
- **工具**：normalize.css（样式重置）

## 使用方法

1. 克隆项目到本地：
   https://github.com/CappuccinoRose/Mobile-HTML-Practice.git
2. 直接用浏览器打开任意HTML文件（如`模拟淘宝查询/index.html`）
3. 修改代码练习：
   - 布局调整：编辑对应HTML/CSS文件
   - 交互逻辑：修改JS文件
   - 样式美化：调整CSS文件

## 练习重点

| 项目名称               | 练习内容                     |
|------------------------|------------------------------|
| 模拟flex布局-携程旅行   | Flex弹性布局、导航栏设计       |
| 模拟rem适配-苏宁易购   | REM单位适配、移动端响应式     |
| 模拟触屏轮播图         | 触屏滑动、自动轮播、小圆点切换 |
| 模拟流式布局-京东购物   | 流式布局、商品卡片排列        |
| 模拟淘宝查询           | JSONP接口调用、搜索建议       |
| 趣味查询网             | 多模块表单、结果展示、交互逻辑 |
